# Stalk-Market
crowdsourcing the best turnip price

### Setup firebase

Install firebase CLI
```
$ npm install -g firebase-tools
```

login firebase

```
$ firebase login
```

Initialize your project

```
$ firebase init
```

# Deploy your site
```
$ firebase deploy --only hosting:stalk-market-4dc91
```

# How to run locally
Python 2
```
$ python -m SimpleHTTPServer
```

Python 3
```
$ python -m http.server
```
